
Input files and script used to plot Figure S6, the average within-meadow GDS  over all GUllmarsfjord and Marstrand sites, using Rozenfeld and SAD genetic distance metrics.

Input files: sum of frequency distributions (GDS) per meadow of all meadows in each fjord area (Gullrmarsfjord or Marstrand), using either Rozenfeld or shared allele distance genetic distance metrics.
- avg_GDS_Gullmars_Rozenfeld.csv
- avg_GDS_Gullmars_SAD.csv
- avg_GDS_Mars_SAD.csv
- avg_GDS_Mars_SAD.csv
